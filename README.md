<h1 align="center">@sveltek/sveltekit-boilerplate</h1>

<p align="center">SvelteKit starter template.</p>

<br>

## What's Included

- Svelte 5
- SvelteKit 2
- TypeScript 5
- ESLint 9
- Prettier 3
- TailwindCSS 4

## Development Setup

Start the local `dev` server:

```sh
pnpm dev
```

## Static Deployment

Build your `static` application:

```sh
pnpm build
```

Run the `preview` command to see a production version:

```sh
pnpm preview
```
